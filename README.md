# Cricket Score Sheet Creator

A professional C-based application for managing and tracking cricket match scores. This tool allows users to create, view, and edit detailed cricket scoresheets directly from the terminal.

## üèè Features

- **Create New Scoresheets**: Record match details including venue, teams, toss information, and individual player performances (runs, balls, wickets, etc.).
- **View Scoresheets**: Access a list of all created matches and view detailed scorecards for each innings.
- **Edit Scoresheets**: Update existing match details and player statistics.
- **Support for Multiple Matches**: Store and manage data for up to 100 matches in a single session.

## üöÄ Getting Started

### Prerequisites

You need a C compiler (like GCC) installed on your system.

### Compilation

Clone the repository and compile the source code:

```bash
git clone https://github.com/usnaveen/Cricket-Score-Sheet.git
cd Cricket-Score-Sheet
gcc "CPP Final Final.c" -o cricket_scoresheet
```

### Running the App

Run the compiled executable:

```bash
./cricket_scoresheet
```

## üìã Usage Guide

1. **Main Menu**: Choose from options to Create, View, or Edit a scoresheet.
2. **Creating**: Follow the prompts to enter match info, batting details for both innings, and bowling details.
3. **Viewing**: Select a match from the list to see its full details.
4. **Editing**: Select a match to modify its information.
